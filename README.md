# Task Manager App 📱

## Features
- Login Screen
- Dashboard with RecyclerView
- Add Task Dialog
- Save Tasks (DataStore)
- Fetch Tasks from API (Retrofit)
- Background Reminder (WorkManager)

## Tech Stack
- Kotlin
- Android SDK
- Retrofit
- DataStore
- WorkManager

## API Used
https://jsonplaceholder.typicode.com/todos
